## Setup Guide üöÄ

### Prerequisites
- Python 3.9+
- Django 4.2.20
- Markdown 3.8
- Requests 2.32.3



### Installation
1. Clone the repo:
   ```bash
   git clone https://github.com/JenpeyChao/chatBot.git
   cd chatBot
   
   ```
2. Install the Requirements
   ```bash
   pip install -r requirements.txt
   ```
3. Run Server
   ```bash
   python manage.py runserver
   ```
